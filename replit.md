# Holiday Rush Backend

## Overview
A simple Express + TypeScript backend API running in ESM mode.

## Project Structure
- `src/index.ts` - Main Express server entry point
- `dist/` - Compiled JavaScript output (generated via build)

## Tech Stack
- Node.js with TypeScript
- Express.js 5.x
- ESM modules

## Development
- Run `npm run dev` to start the development server with hot reload
- Server runs on port 5000

## Build & Production
- `npm run build` - Compiles TypeScript to JavaScript
- `npm start` - Runs the production build

## API Endpoints
- `GET /` - Returns a hello message
